# 🏐 Volleyball Club — Telegram Mini App

Telegram Mini App для запису на групові заняття з волейболу. Підтримує пляжний та класичний волейбол, змагальні та тренувальні формати, відгуки після занять, адмін-панель та інтеграцію з Telegram Bot.

---

## ✨ Функціональність

| Функція | Опис |
|---|---|
| 📅 Календар (2 тижні) | Перегляд і запис на заняття в майбутніх 14 днях |
| 🧙 Майстер запису | Крок за кроком: тип гри → локація → тривалість |
| 👥 Перегляд гравців | Список учасників з їхніми вподобаннями та статистикою |
| 📊 Профіль / Статистика | Відвідано / пропущено / відсоток присутності |
| ⭐ Відгуки в додатку | Оцінка заняття зірками + текстовий коментар |
| 🤖 Telegram Bot опитування | Бот надсилає опитування після кожного заняття |
| ⚙️ Ліміт гравців | За замовчуванням 12, адмін може змінити для кожного дня |
| 🛡️ Адмін-панель | Ліміти, відвідуваність, управління ролями, журнал відгуків |

## 🚀 Запуск

### 1. Встановлення залежностей

```bash
npm install
npm install --prefix frontend
```

### 2. Налаштування `.env`

```env
PORT=3000
APP_URL=http://localhost:3000

# (Необов'язково) токен бота від @BotFather
TELEGRAM_BOT_TOKEN=your_token_here
```

### 3. Зібрати і запустити продакшн сервер

```bash
npm run build   # збирає React у frontend/dist
npm start       # запускає Express + SQLite + Telegram Bot
```

Відкрийте: **http://localhost:3000**

### 4. Розробка (hot reload)

Запустіть два термінали:

```bash
# Термінал 1 — Express API
npm start

# Термінал 2 — Vite фронтенд (з проксі до API)
npm run dev --prefix frontend
```

Відкрийте: **http://localhost:5173**

## 🤖 Налаштування Telegram Bot

1. Відкрийте **@BotFather** у Telegram
2. Надішліть `/newbot` і слідуйте інструкціям
3. Скопіюйте токен у файл `.env`:
   ```
   TELEGRAM_BOT_TOKEN=123456789:ABCDEF...
   ```
4. В BotFather задайте Web App URL (`/setmenubutton`) який вказує на ваш розгорнутий сервер

### Команди бота

| Команда | Дія |
|---|---|
| `/start` | Привітання + кнопка відкрити Web App |
| `/stats` | Особиста статистика відвідуваності |

## 🖥️ Режим симуляції (без Telegram)

Коли додаток відкривається поза Telegram, у верхній частині з'являється **панель симуляції**:

- Швидке перемикання між передвстановленими гравцями та адміністратором
- Кнопка «+ Створити» для генерації нового тестового гравця
- Адміністратор бачить вкладку **Адмінка** у навігації

## 🗄️ API Довідник

### Публічні маршрути

| Метод | Шлях | Опис |
|---|---|---|
| `POST` | `/api/user/auth` | Авторизація / реєстрація за telegram_id |
| `GET` | `/api/sessions` | Список занять на 14 днів |
| `POST` | `/api/sessions/register` | Запис на заняття (з параметрами) |
| `POST` | `/api/sessions/unregister` | Скасування запису |
| `GET` | `/api/user/stats/:userId` | Статистика гравця |
| `POST` | `/api/feedback/submit` | Відправити відгук |

### Адмін маршрути

| Метод | Шлях | Опис |
|---|---|---|
| `GET` | `/api/admin/dashboard` | Всі користувачі + журнал відгуків |
| `POST` | `/api/admin/sessions/capacity` | Змінити ліміт гравців на день |
| `POST` | `/api/admin/attendance` | Відмітити відвідуваність |
| `POST` | `/api/admin/users/role` | Змінити роль (user/admin) |
| `POST` | `/api/admin/sessions/trigger-survey` | Надіслати опитування ботом |
